# Parttime Employee Management Project Context

This folder contains project-specific context for the real course project.

## Important separation

- `project/` is the source of project context for **Parttime Employee Management**.
- `teacher-reference/` is an external hotel-management sample used only for methodology and writing structure.

When project context and teacher-reference differ, project context wins.

## Modules

- `module-1/` — Register for next week: registration staff registers next week's shifts for a part-time employee.

## How to use

Before writing any report phase:

1. Read the relevant files under `project/module-1/`.
2. Verify details against current LaTeX chapters when accuracy matters.
3. Use teacher-reference files only to shape the academic structure, not the project facts.
