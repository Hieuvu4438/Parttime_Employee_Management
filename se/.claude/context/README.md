# Project Claude Context

This folder contains project-local context for Claude Code.

## Context boundaries

- `project/` contains **real project context** for Parttime Employee Management.
- `project/module-1/` contains the detailed context for Module 1: Register for next week.
- `teacher-reference/` contains **external teacher sample materials** for a hotel management system. These files are reference patterns only.
- The real project is **Parttime Employee Management**. Do not copy hotel-specific actors, entities, tables, scenarios, test data, or UI names into the project report.
- When using teacher references, extract the **method and structure** only, then adapt it to the current module.
- For coding tasks, `teacher-reference/06_coding-java-swing-dao.md` is a methodology reference for Java Swing / DAO style, while `project/module-1/07_coding-plan-context.md` contains the real Module 1 coding mapping.

## How to use

1. Read the relevant files under `.claude/context/project/module-1/` for real module context.
2. Read the current LaTeX chapters in `chapters/` to verify the latest project state.
3. Read the relevant teacher-reference phase only to understand the expected academic format or coding pattern.
4. Produce new content using project-specific concepts from Parttime Employee Management only.
