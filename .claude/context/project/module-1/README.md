# Module 1 Context — Register for Next Week

Module 1 of the Parttime Employee Management project is **Register for next week**.

The module supports registration staff in recording a part-time employee's available working shifts for the following week.

- `05_testing-context.md` — testing assumptions and standard black-box test context.
- `07_coding-plan-context.md` — coding context for implementing Module 1 using the teacher's Java Swing / DAO pattern.
